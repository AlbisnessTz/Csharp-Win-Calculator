# C# WinForms Calculator

A simple calculator application built using C# and Windows Forms.

## Features
- Basic operations: +, −, ×, ÷
- Percentage (%)
- Delete last digit
- Clear all
- Decimal point
- Double zero (00)

## Technologies Used
- C#
- Windows Forms
- Visual Studio

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Build and run the project
